highlight uses simple jquery command to syntax highlight R code contained in any regular <pre> element.

For example, this chunk of code, from the datasets::cars

require(stats)
require(graphics)
plot(cars, xlab = "Speed (mph)", ylab = "Stopping distance (ft)", las = 1)
lines(lowess(cars$speed, cars$dist, f = 2/3, iter = 3), col = "red")
title(main = "cars data")
plot(cars, xlab = "Speed (mph)", ylab = "Stopping distance (ft)", las = 1, log = "xy")
title(main = "cars data (logarithmic scales)")

More details is given at highlights page on how to use it on your own page.
